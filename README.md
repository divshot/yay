# yay

Generate random, ridiculous names for anything. Yay!

Examples:

* immense-foodstuffs
* salty-grapefruit
* voiceless-nutritious
* confused-menu

## Install

```
npm install yay --save
```

## Usage

```js
var yay = require('yay');

console.log(yay()); // Outputs something like: immense-foodstuffs
```
